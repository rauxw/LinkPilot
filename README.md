# LinkedIn Cold Outreach Helper

A simple Chrome extension that helps you save LinkedIn profiles and generate personalized cold outreach messages instantly.

## 🚀 Features

- Extracts name from LinkedIn profile tabs
- Validates only real LinkedIn profile URLs (`/in/`)
- Prevents duplicate entries
- Generates personalized cold messages
- Copy-to-clipboard functionality
- Supports custom message input

## 🛠️ Tech Stack

- JavaScript
- Chrome Extension APIs
- LocalStorage

## 📦 How It Works

1. Open a LinkedIn profile
2. Click "Add" to save the lead
3. Generate a cold message
4. Copy and use it directly

## ⚠️ Note

This tool is designed to assist writing messages only. It does not automate sending messages to comply with LinkedIn policies.

Built for learning and improving outreach efficiency.
